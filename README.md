Set of Mudlet aliases to allow creation of Triggers/Aliases etc using the MM2K syntax

### Usage

Implemented MudMaster Commands:

`> /action`
`> /alias`
`> /array`
`> /assign`
`> /call`
`> /chat`
`> /chatall`
`> /chatname`
`> /editvariable`
`> /emoteall`
`> /empty`
`> /event`
`> /if`
`> /itemadd` 
`> /itemdelete`
`> /listadd`
`> /listcopy`
`> /listdelete`
`> /loop`
`> /showme`
`> /unchat`
`> /unvariable`
`> /var`
`> /while`
`> /zap`
