Set of Mudlet aliases to allow creation of Triggers/Aliases etc using the MM2K syntax

### Usage

Implemented MudMaster Commands:

`> /action`
`> /alias`
`> /call`
`> /chat`
`> /chatall`
`> /chatname`
`> /emoteall`
`> /event`
`> /if`
`> /itemadd`
`> /itemdelete`
`> /listadd`
`> /listcopy`
`> /listdelete`
`> /loop`
`> /showme`
`> /unchat`
`> /var`
`> /while`
